# Flujo del prototipo

1. **index.html** — Presentación y crear cuenta / demo.
2. **registro.html** — Dos pasos: datos personales → curso y círculo.
3. **inicio.html** — Hub: estadísticas, carta del día, mini-ruta.
4. **carta.html** — Dos preguntas + resumen → vuelve a inicio.
5. **equipo.html** — Convergencia, energía del círculo y compañeros.

Navegación fija inferior en inicio, carta y círculo.

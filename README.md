# LinguaQuest

Prototipo web de práctica de idiomas para universidad: microlecciones, mapa del curso y meta compartida con el círculo de clase.

## Ver en línea (GitHub Pages)

1. En GitHub, abre **Settings → Pages**.
2. En **Build and deployment → Source**, elige **Deploy from a branch**.
3. Selecciona la rama **main** y la carpeta **/docs**.
4. Guarda. En uno o dos minutos el sitio quedará en:

   **https://selenofilia.github.io/PROYECTO-Duocraft/**

## Estructura del repositorio

```
docs/              Sitio publicado en GitHub Pages
documentacion/     Notas del proyecto (audiencia, flujo, mecánicas)
```

## Desarrollo local

Abre `docs/index.html` en el navegador o sirve la carpeta `docs` con cualquier servidor estático.
